## 先说"搬瓦工不买 CN2"这句话在说什么

## KVM 套餐是什么？它和 CN2 有什么关系

这里有个容易混的地方：KVM 是**虚拟化架构**，不是线路。搬瓦工目前所有套餐底层都是 KVM，所以"KVM 套餐"只是官网对入门套餐的命名方式，全称大概是 20G KVM PROMO 这样的。

这个入门 KVM 套餐的核心特点：

- 年付 $49.99，月流量 1TB
- 可以选 8 个机房，包括洛杉矶 DC2、DC4、DC8，弗里蒙特，新泽西，纽约，荷兰
- 购买后可以在后台（KiwiVM）免费迁移机房
- 支持 CN2 GT 机房（DC3），但迁过去之后**流量变成三分之一**

最后这点很关键。你买了 KVM 套餐，迁到 DC3 CN2 GT 机房，原本 1TB/月的流量就变成 333GB/月。对流量消耗大的用户来说这不合适。

所以有个建议流传很广：**不打算用 DC3 CN2 机房的话，直接买 KVM 套餐，够用且便宜**。
